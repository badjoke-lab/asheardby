# AsHeardBy 実装用UIテキスト一覧 v0.1

## 1. ページ共通

### `<title>`
**AsHeardBy — A comparative listening study**

### メタ説明
聞こえ方の違いについて、一般に言われていること・書かれていることをもとに、ブラウザ上で比較体験向けに表現したサイト。診断ではなく参考表現です。

## 2. Hero

### サイト名
**AsHeardBy**

### サブコピー
**A comparative listening study**

### リード文
聞こえ方の違いについて、一般に言われていること・書かれていることをもとに、ブラウザ上で比較体験向けに表現したサイトです。

### 注意文
これは診断や医療判断のためのものではありません。  
各モードは、比較体験のための参考表現です。

### 補助文
- 音量を低めにして開始してください  
- 一部モードではヘッドホンを推奨します

---

## 3. Source セクション

### セクション見出し
**Choose a source**

### セクション説明
まず音源を選んでください。  
内蔵サンプルのほか、手元の音声ファイルも使えます。

### 内蔵サンプル見出し
**Built-in samples**

### サンプルカード 1
**Voice sample**  
会話や子音の違いを比較しやすい基本サンプル。

### サンプルカード 2
**Noise sample**  
背景雑音の重なりや埋もれ方を確認しやすいサンプル。

### サンプルカード 3
**Stereo sample**  
左右差や定位の偏りを確認しやすいサンプル。

### アップロード見出し
**Upload audio**

### アップロード説明
ローカルの音声ファイルを使って比較できます。

### アップロード補助
音声ファイルはこの端末内で処理されます。

---

## 4. Filter セクション

### セクション見出し
**Choose a listening mode**

### セクション説明
Condition は聞き取り困難や状況差、  
Animal Reference は人間と異なる可聴傾向の参考比較です。

### カテゴリ見出し
**Condition**  
**Animal Reference**

---

## 5. Player / Compare セクション

### セクション見出し
**Listen and compare**

### セクション説明
同じ音源のまま、Original と Filtered を切り替えて比較できます。

### トグル
**Original**  
**Filtered**

### 補助文
- 同じ再生位置のまま切り替わります
- 左右差モードではヘッドホン推奨

---

## 6. Visualization セクション

### 見出し
**What changes**

### 説明
どの帯域が弱くなるか、埋もれるか、または人間の可聴域外にあるかを、簡略化した表示で比較します。

### 主表示ラベル
- **Low**
- **Low-mid**
- **Speech**
- **High**
- **Very high**

### 補助表示ラベル
- **Left / Right balance**
- **Noise overlap**
- **Human range**
- **Extended high**
- **Extended low**

## 8. About / Notes セクション

### 見出し
**About this project**

### 本文
AsHeardBy は、聞こえ方の違いについて一般に説明される特徴をもとに、比較体験向けの形に翻訳して提示するプロジェクトです。  
個人の聞こえ方を正確に再現するものではなく、診断を目的とするものでもありません。
